Technical Competition Highlights

Accenture UIC Competition: Project A.C.T

AI for Social Good Public Service

We addressed the strain on Singapore's CDC voucher support. Project A.C.T uses AI-assisted support to handle routine enquiries, prioritizing cases for officers to deliver trusted help while preserving human care and accessibility.

Microsoft Competition: AbleConnect

Inclusive Tech AI Personalization

Created AbleConnect, a platform connecting persons with disabilities to jobs using AI for role recommendations and adaptive resume generation. Includes a live accessibility checker for inclusive hiring.

SMU Hackathon: Project SLAA

Generative AI SkillsFuture Integration

Developed an AI-driven "Resume Builder" chatbot that identifies skills gaps and provides personalized SkillsFuture course recommendations, integrated with Singpass for secure access.
